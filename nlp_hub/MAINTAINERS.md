## Responsibilities

### Maintainer Duties

- **Code Reviews**: Review pull requests and maintain code quality standards
- **Issue Triage**: Manage and prioritize issues and feature requests
- **Release Management**: Publish releases and manage versions
- **Documentation**: Update and maintain project documentation
- **Community Support**: Respond to discussions and help contributors

### Decision Making

- Major architectural changes require maintainer approval
- Features should be discussed before implementation
- Security issues should be reported privately
